Exchange Server for Ethereum and Algorand Tokens Exchange 

Build an exchange server (“custodial exchange”) for processing user buy and sell orders for tokens on 
the Ethereum and Algorand Blockchains. 

— Built a server with Python-Flask to receive user requests for exchange.Verified, matched, and filled 
orders on Algorand PureStake and Ethereum TestNet. 
— Used SQL Alchemy for database operations (to store received orders, derived/child orders and 
executed transaction in database tables).
